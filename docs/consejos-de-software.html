3.6 Consejos de software

El software que recomendé para el manejo de notas puede ser más que suficiente para manejar información de forma adecuada. En verdad son muy útiles y tienen lo necesario para trabajar.22 Aun así hay todavía más opciones para organizar tu información a través de entornos más complejos, aunque no por eso son mejores. Algunas que te podrían interesar son: Trilium, una base de conocimiento personal con organización multi-jerárquica, software libre y abierto; Foam, una versión libre de Roam Research basado en Visual Studio Code; Obsidian, un entorno de conocimiento local de sintaxis Markdown con plugins y API abierta; TiddlyWiki, un cuaderno digital no-lineal para capturar y organizar información en wikis personales; Zettlr, un editor Markdown preparado para el sistema zettelkasten e integración con gestores de referencias. Otras opciones son utilizar entornos de desarrollo integrado como Atom o el potente, pero complicado Org-mode para Emacs, entre otras. Las características que ofrecen estas herramientas son más especializadas y particulares, por lo que implican cierta curva de aprendizaje para su uso. Puede ser un objetivo a mediano plazo conocer sobre ellos para después integrarlos a tu proceso de trabajo. Ten en cuenta que las migraciones de software son un dolor de cabeza, es mejor trabajar en un entorno estable para ti y para tu trabajo.
