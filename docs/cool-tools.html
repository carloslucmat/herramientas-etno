Sección 5 Cool tools

Software para análisis cualitativo

CAQDAS FLOSS

  • QualCoder - Software de análisis y codificación escrito en Python con soporte para texto, imágenes, audio y video. Libre, abierto y gratuito.
  • QCoder - Paquete para la codificación de datos en R. Libre, abierto y gratuito.
  • Taguette - Software para codificación de datos de texto. Abierto y gratuito con aplicación Web y de escritorio.

Gestores de bibliotecas personales

zot-jab-men

  • Zotero - Gestor de referencias libre, abierto y gratuito.
    • Better BibTex for Zotero - Integración de BibLaTex para Zotero.
    • ZotFile for Zotero - Gestión de archivos PDF para Zotero.
    • Plugins desarrollados por la comunidad.
    • Zotero hacks - Pequeños trucos para usar Zotero.
  • JabRef - Gestor de referencias bibliográficas de código abierto.
  • Mendeley - Gestor bibliográfico y red social académica de Elsevier.
  • Calibre - Un administrador de libros electrónicos.

Programas para tomar notas

  • Joplin - Software libre, de código abierto para tomar notas y tareas pendientes en Markdown para escritorio y celular.
  • Notion - Aplicación todo-en-uno para tomar notas, planear proyectos, crear tableros, calendarios, recordatorios y manejar información.
  • Simplenote - Aplicación de notas minimalista multiplataforma, con respaldo en la nube, sincronización automática, soporte Markdown y sostenido por el quipo de WordPress y Tumblr.
  • Notable - Aplicación de notas en Markdown basado en VS Code.
  • WorkFlowy - Una aplicación web para la toma de notas en outlines al estilo fractal.
  • RemNote - Notas jerárquicas y aprendizaje con repaso espaciado.
  • Boostnote -Espacio de trabajo colaborativo Markdown con herramientas para programadores.
  • Ghostwriter - Editor Markdown limpio y sin distracciones para Linux y Windows.

Editores de texto para escribir (y otras cosas)

  • Atom - Un gran editor de texto extensible para programar, escribir, seguir tus proyectos en GitHub y mucho más.
    • Atom para el flujo de trabajo académico - Presentación de Juan Carlos Castillo para iniciados en Atom.
    • Using Atom for academic writing - Paquetes de Atom para la escritura académica.
  • WriteMonkey - Una aplicación zenware para escribir sin distracciones.
  • Doom Emacs - Una configuración para domar al imponente Emacs.
  • LibreOffice - Un reemplazo libre y gratuito de la suite Office.
  • RStudio - Entorno de desarrollo integrado para el lenguaje de programación R. Aunque generalmente se usa para ciencia de datos y estadística, también tiene otras cosas como R Markdown, Bookdown, RQDA y otros complementos interesantes.
  • Marktext - Un editor Markdown simple y elegante, centrado en la velocidad y la usabilidad.
  • Zettlr - Editor Markdown basado en el sistema Zettelkasten, pensado para académicos y con integración para gestores de referencias.

Pandoc

  • Pandoc - La navaja suiza para la conversión de documentos creada por el filósofo John MacFarlane.
  • Pandoc Starter - Un repositorio con varios archivos para comenzar con Pandoc.
  • minimal make - Un mínimo tutorial sobre los makefiles para automatizar los procesos de Pandoc y utilizar plantillas LaTeX.
  • Academic Pandoc Template - Una sencilla plantilla académica para artículos académicos.
  • Pandoc Scholar - Plantilla académica completa con metadatos mejorados.
  • Arabica - Plantilla para libros académicos con Pandoc y LaTeX.
  • Plantillas pandoc aportadas por usuarios.
  • Eisvogel - Una plantilla académica sencilla de pandoc LaTeX.

Escritura académica con Markdown + Pandoc

A plain-text document toolchain en el libro de Kieran Healey.

  • Sustainable Authorship in Plain Text using Pandoc and Markdown - Un tutorial con lo que tienes que saber sobre la escritura en texto plano con Markdown y Pandoc.
  • Escritura científica con Markdown - Una guía de Jaan Tollander de Balsch.
  • Markdown and Pandoc for academic writing - Una pequeña introducción a las ventajas de Markdown y Pandoc en el trabajo académico. Incluye una plantilla YAML.
  • The YAML Fieldguide - Una útil guía sobre YAML para tus documentos.
  • A Plain Text Workflow for Academic Writing with Atom - Un flujo de trabajo que puede ayudarte a hacer el tuyo.
  • Citando en texto plano - Tuturial de Juan Carlos Castillo.
  • Acerca de escribir y publicar un libro con Bookdown, GitHub y Zotero por Jack Dougherty y compañía.
  • scientific-markdown - Example for use of Markdown for scientific publications using Pandoc and latexmk.
  • Template for writing a PhD thesis in Markdown - Clean organization of files to provide a framework for writing a PhD thesis in mostly Markdown with a little bit of LaTeX, and compiled with Pandoc.
  • thesisdown - Un proyecto inspirado en bookdown para hacer tu tesis con R Markdown. Incluye plantillas de distintas Universidades.
