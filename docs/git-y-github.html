  • The Plain Person’s Guide to Plain Text Social Science - Pequeña guía de Kieran Healy sobre el flujo de trabajo académico en texto plano para ciencias sociales. Contenido sobre R Markdown, editores de texto y hasta una una introducción a Emacs.

    • Preserving Your Research Data - Para mantener consultables tus datos en el tiempo.
    • Fetching and Parsing Data from the Web with OpenRefine - Para explorar, limpiar y transformar datos de la Web.
    • Running a Collaborative Research Website and Blog with Jekyll and GitHub - Para construir tu página colaborativa de investigación.
  • Investigación Social Abierta: Herramientas para la reproducibilidad, colaboración y comunicación académica_ Libro digital de Juan Carlos Castillo.

    • Writing for an academic journal: 10 tips.
    • How to write a scientific abstract in six easy steps.
    • Harvard Writing Resources.
